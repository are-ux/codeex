# LedgerLink

LedgerLink is a responsive B2B SaaS landing page built with Next.js, Tailwind CSS, and Framer Motion.

## Getting started

### Prerequisites
- Node.js 18+
- npm

### Install dependencies
```bash
npm install
```

### Run the dev server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
